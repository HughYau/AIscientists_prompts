# AIscientists_prompts

可搜索、可筛选的 AI-Scientist 提示词画廊，支持一键复制与阅读指引，适合研究与写作工作流。

- 在线预览: <https://hughyau.com/AIscientists_prompts/>
- 仓库地址: <https://github.com/HughYau/AIscientists_prompts>
- Star 一下支持项目发展！

![GitHub Repo stars](https://img.shields.io/github/stars/HughYau/AIscientists_prompts?style=social)

## 功能

- 分类筛选 + 关键词搜索
- 结构化卡片展示，包含用途、占位符、来源与许可
- 一键复制（含归属说明，便于合规引用）
- 阅读指引模态框（输出规范、速用模板、许可提示）

## 本地预览

无需构建，直接打开 `index.html` 即可。也可使用任意静态服务器（如 VS Code Live Server）。

## 如何贡献

欢迎通过 Issue/PR 贡献提示词与改进：

- 先阅读贡献指南：[`CONTRIBUTING.md`](./CONTRIBUTING.md)
